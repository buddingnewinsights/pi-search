# v0.1.0

Initial standalone release of **pi-search**.

## Included tools

- `websearch` — web search via Exa AI
- `codesearch` — code/documentation search via Exa AI
- `grepsearch` — real-world GitHub code search via grep.app

## Notes

- packaged as a standalone pi extension
- scoped package publish configured as public
- repository mirrored to both `buddingnewinsights/pi-search` and `heyhuynhgiabuu/pi-search`
